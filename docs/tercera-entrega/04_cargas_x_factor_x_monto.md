# 04 — Cargas X Factor y X Monto

## Alcance futuro

Las cargas X Factor y X Monto se implementarán posteriormente para procesar archivos tributarios definidos por el proyecto.

## Estado actual

Aún no existen endpoints, archivos, plantillas, almacenamiento, validaciones ni pantallas asociadas a estas cargas. Los archivos futuros deberán quedar fuera de carpetas públicas.
